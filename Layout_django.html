How to install Django ?

By Naman Jain | 6 Minute read |

We know django is a python frame work , so first we assure that django is installed in our systems. Let's check it...

open window shell or command prompt in your window and type following command

python --version

If you already have Python installed, then check its version is above or equal to 3.5.

If python is not installed in your PC then go to this site www.python.org/downloads/

Pip is a Python Package Manager. It makes installing and uninstalling python package very easy. Pip is pre-installed in your system with python for above 3.4 version.

If pip is not pre-installed then visit https://pip.pypa.io/en/latest/installing/
